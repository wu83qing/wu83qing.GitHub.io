Foundation框架

发表于 2017-06-07 |

官方文档链接

Foundation 框架为应用程序和框架提供了基础层功能,包括数据存储和持久性,文本处理,日期和时间计算,排序和过滤以及联网. 由Foundation定义的类,协议和数据类型在整个macOS,iOS,watchOS和tvOS SDK中使用.

  • Fundamentals
    • Numbers,Data, and Basic Values
      使用Cocoa中使用的原始值和其他基本类型
    • Strings and Text
      创建并处理Unicode字符串,使用正则表达式查找模式,并对文本进行自然语言分析.
    • Collections
      使用数组,字典,集合和专用集合来存储和迭代对象或值的组.
    • Dates and Times
      比较日期和时间,并执行日历和时区计算.
    • Units and Measurement
      使用物理尺寸标记数字数量以允许在相关单元之间进行区域设置感知的格式设置和转换.
    • Data Formatting
      将数字,日期,度量值和其他值转换为区域感知字符串表示形式.
    • Filters and Sorting
      使用断言,表达式和排序描述符来检查集合和其他服务中的元素.
  • App Support
    • Task Management
      管理您的应用程序的工作以及与用户和系统的交互.
    • Resources
      Access assets and other data bundled with your app.
    • Notifications
      用于广播信息和订阅广播的设计模式.
    • App Extension Support
      管理应用扩展程序与其托管应用程序之间的交互.
    • Errors and Exceptions
      回应与API交互中的问题情况,并对您的应用进行微调,以便更好地进行调试.
    • Scripting Support
      允许用户使用AppleScript和其他自动化技术控制您的应用程序,或从您的应用程序中运行脚本
  • Files and Data Persistence
    • File System
      创建,读取,写入和检查文件系统中的文件和文件夹.
    • Archives and Serialization
      将对象和值转换为属性列表,JSON和其他 flat binary 表示.
    • Preferences
      持久保存用于配置应用程序的域范围信息.
    • Spotlight
      搜索本地设备上的文件和其他项目,并索引您的应用内容以进行搜索.
    • iCloud
      管理在用户的iCloud设备之间自动同步的文件和键值数据.
  • Networking
    • URL Loading System
      与URL进行交互并使用标准Internet协议与服务器进行通信.
    • Bonjour
      广告服务,方便在本地网络上发现,或发现其他人宣传的服务.
  • Low-Level Utilities
    • XPC
      管理安全的进程间通信.
    • Object Runtime
      获得基本的Objective-C功能,Cocoa设计模式和Swift集成的低级支持.
    • Processes and Threads
      管理应用程序与主机操作系统和其他进程的交互,并实现低级别的并发功能.
    • Streams,Sockets,and Ports
      使用低级Unix功能来管理文件,进程和网络之间的输入和输出.
  • Reference
    • Foundation Constants
    • Foundation Data Types
      本文档描述了Foundation框架中的数据类型和常量.
